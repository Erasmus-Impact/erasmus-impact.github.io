Example Speaker

This is a subtitle

After copying this file or its content and adjusting the front matter above, everything below this line (and this line aswell) can be deleted and replaced with content about the author.

The new file should be named firstname-lastname.md, all lowercased.

This section of front matter should be copied to the beginning of an author file (every markdown file in content/en/author/). Replace all relevant values with the information specific to the author, and remove unused options. Recall front matter (the definition block) has to be enclosed with single lines containing ---, also lines in this block starting with or containing a # will be treated as comments and take no effect from the position of the #. The only exception to this is when the # is inside quotes, for example in links.

---
# Title: this should be the name of the author
title: "Example Speaker"

# Name: should be the same as the title, will be displayed below the image or avatar
name: "Example Author"

# Description: this is more a subtitle than a description, displayed directly below the name.
description: "This is a subtitle"

# Email: The email to contact the author.
# Will be displayed below the photo/identicon
email: "example@example.com"

# Photo: An image of the author to be displayed on the website.
# If no photo is given, and if an email is given, gravatar will be used to either get an image from there or put a generated unique placeholder, similar to github profile pictures
#photo: "images/author/example.jpg"

# bg_image: An optional image which will be shown in the page header, behind the title and description. Will have a transparent gray overlay to ensure readability.
bg_image: "images/feature-bg.jpg"

# Social: A list of social or other icons with links. For icon, any icon listed at https://ionicons.com/v2/ can be used. A list entry starts with a dash.
social:
  # A list entry with a github icon.
  - icon : "ion-social-github-outline"
  # The link the above icon will direct to. Will be on this page, unless a protocol (https:// or http://) is included.
    link : "#"
  
  # Another list entry with a linkedin icon. 
  - icon : "ion-social-linkedin-outline"
  # The link the above icon will direct to. Will be on this page, unless a protocol (https:// or http://) is included.
    link : "#"

# Whether this page is a draft. Draft pages will only be displayed in the draft preview (erasmus-impact.github.io), not on the public site (impact.uni-bremen.de)
draft: true
---
