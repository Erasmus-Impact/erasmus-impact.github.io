Choose an appropriate name for the news file, for example example-partner.md.

This section of front matter should be copied to the beginning of a partners file (every markdown file in content/en/partners/, except _index.md). Replace all relevant values and remove unused options. Recall front matter (the definition block) has to be enclosed with single lines containing ---, also lines in this block starting with or containing a # will be treated as comments and take no effect from the position of the #. The only exception to this is when the # is inside quotes, for example in links.

---
# title: The title or name of the project partner. 
#        Gets displayed on the list of partners and the partner site.
title: "Example Partner"

# description: The role of the project partner. Can be omitted.
#              Gets displayed on the partner site below the title.
description: "Example Role in Project"

# image: The logo of the partner.
#        Gets displayed on the list of partners and the partner site.
image: "images/partners/example-logo.png"

# homepage: The URL to the homepage.
#           The logo on the partner site will link there.
homepage: "https://example.com"

# contact: Contact information.
#          Gets displayed on the partner site.
#          Sub-level parameters person, phone, email have to be indented to the same depth.
contact:
# person: The person someone is contacting with the information
  person: "Example contact person Name"
# phone: The phone number of the person
  phone: "+49 111 000 111 000"
# email: The email address of the person
  email: "example@example.com"

# gallery: When this parameter is set, a slideshow of images will be shown on the partner site.
#          The slideshow is shown beteween the contact information and the additional content.
#          To specify the images, simply set multiple values for the sub-parameter image by starting with a dash.
# gallery:
     # image: The images to show in the slideshow.
#   - image: "images/partners/example/1.png"
#   - image: "images/partners/example/2.png"

# gallery_folder: As an alternative to the parameter gallery, this parameter can be used to achieve the same slideshow
#                 All images located in that folder will be used.
#gallery_folder: "images/partners/example/"

# Aliases: For partners, at least one alias to reach this page should be set.
aliases:
  # Mandatory alias to allow partners to be set as speakers, authors or organisators
  - /author/example-partner

# Whether this page is a draft. Draft pages will only be displayed in the draft preview (erasmus-impact.github.io), not on the public site (impact.uni-bremen.de)
draft: true
---
