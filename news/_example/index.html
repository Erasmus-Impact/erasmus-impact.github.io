After copying this file or its content and adjusting the front matter above, everything below this line (and this line aswell) can be deleted and replaced with the content of the news.

Choose an appropriate name for the news file, for example news-title.md.

This section of front matter should be copied to the beginning of a news file (every markdown file in content/en/news/, except _index.md). Replace all relevant values and remove unused options. Recall front matter (the definition block) has to be enclosed with single lines containing ---, also lines in this block starting with or containing a # will be treated as comments and take no effect from the position of the #. The only exception to this is when the # is inside quotes, for example in links.

---
# Title: The title of the News
title: "Example News"

# Description: A description of the News. Will be shown below the title.
description: "This is an example news page."

# Date: The date the news is from. If this date is in the future, the news won't be shown on the website until that date is reached and the website is generated.
date: 2021-07-20T18:00:00+02:00

# PublishDate: If date is in the future, but the news should be shown anyways, set this to the date when the news shall be published. Otherwise omit.
publishDate: 2021-07-20T16:00:00+02:00

# Author: The author of the News. Will be shown on the news page, and will link to the corresponding author description page in author/.
author: Moritz Bergenthal

# Tags: A list ("array") of tags that will get assigned to the news. News can be filtered by tags.
tags: ["Announcement", "Event"]

# Categories: A list ("array") of categories the news will get assigned to. News can be filtered by category.
categories: ["Job offers"]

# bg_image: An optional image which will be shown in the page header, behind the title and description. Will have a transparent gray overlay to ensure readability.
bg_image: "images/feature-bg.jpg"

# Type: For news, this should always be post. Do not change.
type: "post"

# Whether this page is a draft. Draft pages will only be displayed in the draft preview (erasmus-impact.github.io), not on the public site (impact.uni-bremen.de)
draft: true
---
