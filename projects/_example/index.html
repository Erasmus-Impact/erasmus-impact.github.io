Choose an appropriate name for the news file, for example project-title.md.

This section of front matter should be copied to the beginning of a news file (every markdown file in content/en/projects/, except _index.md). Replace all relevant values and remove unused options. Recall front matter (the definition block) has to be enclosed with single lines containing +++, also lines in this block starting with or containing a # will be treated as comments and take no effect from the position of the #. The only exception to this is when the # is inside quotes, for example in links.

Be aware that project pages require the fron matter to be TOML, that means instead of the usual enclosing --- pages here need enclosing +++, also parameter assignment is done by parameter="Value" instead of parameter: "Value".

+++
# Title: The title of this project or event
title="Example Project"

# Description: A short description or subtitle of the project. Can also be the series the event is from.
description="Subtitle of the Project"

# Startdate: The date and time the event or project starts.
startdate=2021-06-10T11:00:00

# Enddate: The date and time the event or project ends. Can be omitted.
enddate=2021-06-10T17:00:00

# Speaker: A list of speakers, author, organisators or other relevant persons. It will be tried to link to /author/person.
speaker=["University of Girona", "Eduardo Ochoa", "Miguel Castillón"]

# Image: An image for the project, will be displayed in the list of projects
image="images/projects/example"

# Icon: When no image is set, an ion-icon (see https://ionicons.com/v2/) can be set as an alternative.
icon="ion-alert"

# bg_image: An optional image which will be shown in the page header, behind the title and description. Will have a transparent gray overlay to ensure readability.
bg_image="images/feature-bg.jpg"

# EventCategories: A list ("array") of categories the project gets associated with. Projects can be filtered by categories. For example, this can be the series the event is from.
eventcategories=["Webinar", "Coffee with EUMR"]

# Aliases: A list ("array") of aliases that link to this page. Can be used to fix author/speaker links for persons that don't have their own author/speaker page.
aliases=["/author/miguel-castillón", "/author/eduardo-ochoa"]

# Type: For projects, events or other pages here, this should always be event.
type="event"

# Whether this page is a draft. Draft pages will only be displayed in the draft preview (erasmus-impact.github.io), not on the public site (impact.uni-bremen.de)
draft=true
